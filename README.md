### Golang + MySQL + Gin + Gorm 

Requirements: Go, [Docker](https://www.docker.com/)

Steps:

```
# Clone the project
go get github.com/gpuenteallott/golang-mysql-docker-setup

# Run containers
docker compose build
docker compose up -d

# Confirm it all went up correctly. Exit with Ctrl+C
docker compose logs
```
